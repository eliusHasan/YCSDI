# Youth Career & Skill Development Institute

Separate MERN stack project structure.

## Folders

```txt
frontend/   React + TypeScript frontend
backend/    Node.js + Express.js + TypeScript backend
uploads/    Local upload folders
docs/       Project documentation
```

## Tech Stack

- Frontend: React, TypeScript, Vite, Tailwind CSS
- Backend: Node.js, Express.js, TypeScript
- Database: MongoDB with Mongoose
- Authentication: Firebase login on the frontend, Firebase Admin token verification on the backend

## Development

Install dependencies separately:

```bash
npm install --prefix frontend
npm install --prefix backend
```

Run apps separately:

```bash
npm run frontend
npm run backend
```
