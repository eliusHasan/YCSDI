# MongoDB Database Architecture

The backend uses MongoDB with Mongoose.

Recommended folder locations:

- Connection setup: `backend/src/database`
- Mongoose models: `backend/src/models`
- Feature-specific business logic: `backend/src/modules`
- Request validation: `backend/src/validators`
